# sst

SST is short for static site. It compiles Jinja2 formatted templates, markdown, and HTML into a directory that can be served statically.

```
.
├── pages
│   ├── index.html
│   └── about.md
├── posts
│   └── first-post.html
├── public
│   ├── my-picture.jpg
│   ├── manifest.xml
│   └── favicon.ico
└── templates
    ├── header.html.tpl
    ├── footer.html.tpl
    ├── base.html.tpl
    └── post.html.tpl
```
